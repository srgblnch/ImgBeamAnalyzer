Image Analyzer
Device Description

ImgBeamAnalyzer Class

Revision: - Author:
The device analyzes images accessible on another remote device, and proposes the following (selectable) features : - in preprocessing : rotation, mirroring, gamma correction - ROI (Region Of Interest), either user-defined or automatic by blob analysis - image moments (mean intensity, centroid, variance, skew) - maximal intensity (to monitor saturation for example) - ellipse fitting when automatic ROI is desired - profiles with corresponding gaussian fit - 2D gaussian fit of the image
